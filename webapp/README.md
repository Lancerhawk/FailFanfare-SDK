# FailFanfare Documentation Webapp

This is the interactive documentation site for the FailFanfare SDK. Built with **React 19**, **Vite 7**, and **TypeScript**, it's a fun way to explore how the SDK handles your dev workflow.

## Features

- **Live Configuration Reference**: Exhaustive mapping of all SDK options and implementation details.
- **Sound Event Guide**: Technical triggers and fallback behaviors for every auditory event.
- **Micro-Particle Background**: Bespoke floating particle system using Vanilla CSS animations.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop viewports.

## Development

To run the documentation site locally:

1.  **Navigate to the folder**:

    ```bash
    cd webapp
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Launch dev server**:
    ```bash
    npm run dev
    ```

## Structure

- `src/App.tsx`: Main application logic and documentation content.
- `src/App.css`: Premium design system, glassmorphism, and animations.
- `src/main.tsx`: Entry point.
